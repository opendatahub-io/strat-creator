# Dashboard Guide

The [strat-dashboard](https://strat-dashboard-0f1209.gitlab.io/) provides aggregate statistics across pipeline runs.

## What This Repo Controls

The dashboard data is generated by `scripts/generate-dashboard.py` in this repo. It produces:

- **`summary.json`**: Aggregated statistics across all runs
- **`runs/<timestamp>.json`**: Per-run data with individual strategy scores

This data is published to the `strat-pipeline-data` repo (GitLab) and served by the `strat-dashboard` site (GitLab Pages).

## Key Metrics

### Executive Summary

- **Total strategies processed**: Count of unique RHAISTRAT tickets across all runs
- **Rubric-pass rate**: Percentage of strategies that scored APPROVE on first evaluation
- **Needs-attention count**: Strategies currently awaiting human intervention
- **Signed-off count**: Strategies with `strat-creator-human-sign-off`

### Per-Run Data

Each pipeline run produces a timestamped JSON file with:

- Strategy keys processed in this run
- Per-strategy scores (feasibility, testability, scope, architecture)
- Per-strategy verdicts (APPROVE/REVISE/REJECT)
- Prose review summaries

### Scoring Trends

The dashboard shows how scores change over iterations. A strategy that was REVISE in run 1 and APPROVE in run 2 indicates successful refinement.

## Accessing Dashboard Data

### Web UI

Visit [strat-dashboard-0f1209.gitlab.io](https://strat-dashboard-0f1209.gitlab.io/) for the visual dashboard.

### JSON API

```bash
# Aggregate summary
curl https://strat-dashboard-0f1209.gitlab.io/summary.json

# Specific run (replace timestamp with an actual run ID)
curl https://strat-dashboard-0f1209.gitlab.io/runs/<TIMESTAMP>.json
```

## Dashboard vs. Jira

| Question | Where to Check |
|----------|---------------|
| What's the verdict for one strategy? | Jira (check RHAISTRAT labels) |
| How are all strategies trending? | Dashboard (scoring trends) |
| What happened in the last pipeline run? | Dashboard (latest run data) |
| Is a strategy signed off? | Jira (`strat-creator-human-sign-off` label) |
